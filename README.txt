Для запуска сервера нужно выполнить ChatMessengerServer.java, который находиться Messenger\src\main\java\server
Для запуска клиента для пользователя нужно выполнить ChatMessengerAppl.java, который находится Messenger\src\main\java\client
Для запуска клиента сервера нужно выполнить CharMessengerServerApplication.java, который находится Messenger\src\main\java\client\server_client
Также в Messenger\out\artifacts\Messenger_jar находится файл Messenger.jar с исполняемым классом ChatMessengerAppl
